# 🎉 Converta Tudo! - Conversor de Arquivos Online

Um conversor de arquivos **rápido**, **fácil** e **100% gratuito** que roda completamente no seu navegador!

## ✨ Características

- 🖼️ **Conversão de Imagens** - JPG, PNG, WebP, GIF e mais
- 🎬 **Conversão de Vídeos** - MP4, WebM, AVI (em desenvolvimento)
- 📄 **Conversão de PDFs** - Criar, editar, mesclar (em desenvolvimento)
- ⚡ **Rápido** - Processamento local, sem uploads
- 🔒 **Seguro** - Seus arquivos nunca saem do navegador
- 💰 **Gratuito** - Sem limite de conversões
- 📱 **Responsivo** - Funciona em qualquer dispositivo
- 🌙 **Interface Limpa** - Design moderno e intuitivo

## 🚀 Começar Rápido

### Desenvolvimento Local

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/converta-tudo.git
cd converta-tudo

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

### Build para Produção

```bash
npm run build
npm run preview
```

## 🏗️ Tecnologias

- **Frontend**: React 18 + TypeScript
- **Build**: Vite
- **Styling**: Tailwind CSS
- **Conversão**: Canvas API, FFmpeg.wasm, PDF-lib
- **Hosting**: GitHub Pages
